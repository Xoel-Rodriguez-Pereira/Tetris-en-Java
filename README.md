## 📖 Descripción

- Es un proyecto de una tarde que utilicé para aprender a usar la librería JFrame.

---

## ✨ Características

- Todos los tetraminós implementados
- El sistema de puntuación está basado en el original de Tetris
- Efectos de sonido para distintas acciones
- Botón de pausa

---

## 🛠️ Tecnologías utilizadas

- Java 21
- Gradle
- JUnit 4 (No hay casos test)
- Visual Studio Code (Con la extensión Java de Oracle y el "Extension Pack for Java" de Microsoft)
- Git

---

## 📦 Requisitos

- Java JDK 21 o superior
- Git

---

## 🚀 Instalación

```bash
git clone https://github.com/usuario/proyecto.git

cd proyecto

```
Si utilizas gradle con eso es suficiente.

Si utilizas maven necesitas convertir el archivo gradle.build en pom.xml con alguna herramienta .

---

## ▶️ Uso

Para ejecutar el programa situate en la carpeta raiz del proyecto con:

```bash
cd nombre-del-directorio/
```

y ejecuta en terminal:

```bash
java -jar /app/build/libs/app-1.0-SNAPSHOT.jar
```
